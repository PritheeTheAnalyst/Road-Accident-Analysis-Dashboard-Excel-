## Project Description
This project was built using Microsoft Excel to help stakeholders analyze road accident data for the years 2021 and 2022. The objective was to transform raw accident records into a clear, interactive dashboard that provides actionable insights into casualties, severity, vehicle involvement, road conditions, and location patterns.

The final output is a fully interactive Excel dashboard designed for decision-makers.

## Purpose of the Project
The client needed a Road Accident Dashboard to:
- Understand the scale and severity of road accidents
- Identify high-risk vehicles, road types, and conditions
- Compare accident trends between 2021 and 2022
- Support data-driven road safety and policy decisions

## Questions (KPIs) 
### Primary KPI Question
- What is the total number of casualties resulting from road accidents in 2021 and 2022?
### Accident Severity (Primary KPI Breakdown)
- How are total casualties distributed by accident severity (Fatal, Serious, Slight)?
- What percentage of total casualties does each accident severity represent?
- Which vehicle type records the highest number of casualties?
### Secondary KPI Questions
- How do total casualties vary by vehicle type?
- How do monthly casualty trends compare between the current year and the previous year?
- Which road types record the highest number of casualties?
- How are total casualties distributed across different road surface conditions?
- How do casualties differ by area (Urban vs Rural)?
